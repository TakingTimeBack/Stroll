📱 Install Stroll as an App

What is a PWA (Progressive Web App)?

Stroll can be installed on your phone like a regular app - no App Store needed!

✅ Benefits:

  • 💾 Works offline (cached version)
  • ⚡ Loads faster than browser
  • 📱 Full-screen, like native app
  • 🔔 Can send notifications
  • 🎯 Home screen icon
  • 📱 No App Store needed

How to install:

iPhone: Tap Share → Add to Home Screen
Android: Tap menu (⋮) → Install App or Add to Home Screen
Desktop: Click the install icon in your browser address bar
